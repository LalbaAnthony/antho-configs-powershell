# Configs PowerShell

Useful PowerShell aliases and functions to boost your productivity in the terminal.

## 🚀 Setup

Empty `$PROFILE` if needed:
```ps1
$null > $PROFILE
```

Install:
```sh
irm https://raw.githubusercontent.com/LalbaAnthony/antho-configs-powershell/main/install.ps1 | iex
```

Update:
```sh
irm https://raw.githubusercontent.com/LalbaAnthony/antho-configs-powershell/main/install.ps1 | iex
```

Uninstall:
```sh
irm https://raw.githubusercontent.com/LalbaAnthony/antho-configs-powershell/main/uninstall.ps1 | iex
```
